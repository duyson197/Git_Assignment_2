# backlog.md

## Product Backlog

1. As a user, I want to create an account, so that I can access personalized features.

2. As a user, I want to log into the system, so that I can securely access my data.

3. As an admin, I want to manage user accounts, so that I can control system access.

4. As a user, I want to reset my password, so that I can recover access if I forget it.

5. As a developer, I want Git branches for each feature, so that team members can work without conflicts.
